# make 동사 대화 대본 (리스닝용)

> 출처 표현: **김재우 기본동사100 — DAY 22 MAKE① / DAY 23 MAKE② / DAY 24 MAKE③**
> 상황: 오랜만에 만난 친구 **Emma(여)** 와 **Jake(남)** 의 수다.
> 약 2분 30초 분량. 실제 대화체 + 축약형(contraction) 위주라 연음이 자연스럽게 들립니다.

| 화자 | 영어 | 한국어 | 쓰인 make |
|---|---|---|---|
| Emma | Hey Jake! Long time no see. How's work going? | 제이크! 오랜만이야. 회사는 어때? | |
| Jake | Oh, it's rough. My boss **makes me upload** new video content every single week. | 아, 힘들어. 상사가 매주 새 영상 올리라고 시켜. | ③ 강제 |
| Emma | Ugh, that sounds exhausting. At least you're eating well, right? Last time you said you were learning to cook. | 으, 지치겠다. 그래도 잘 챙겨 먹지? 저번에 요리 배운다며. | |
| Jake | Yeah, actually! I **made myself go** to a cooking class. I really didn't want to, but I'm glad I did. | 응! 억지로 요리 수업 갔거든. 진짜 가기 싫었는데 가길 잘했어. | ③ 강제(myself) |
| Emma | Nice. So can you actually **make** anything now? | 오, 그래서 이제 뭐 만들 줄 알아? | |
| Jake | Honestly? I **can make you** steamed galbi. My mom used to **make us** that every Sunday. | 솔직히? 갈비찜 해 줄 수 있어. 엄마가 일요일마다 해 주시곤 했거든. | ① 만들어 주다 |
| Emma | No way, that's my favorite. Okay, you've gotta **make me** some. | 진짜? 그거 내 최애인데. 나 좀 해 줘야 해. | ① |
| Jake | Deal. So what about you? How's the new relationship going? | 콜. 넌 어때? 새 연애는 잘돼 가? | |
| Emma | It's going really well. He's so thoughtful. Look, he **made me** this bracelet from string. | 완전 잘돼. 진짜 자상해. 봐, 끈으로 이 팔찌 만들어 줬어. | ① |
| Jake | Aw, that's cute. Wait, is he already **making you wear** matching outfits? | 오, 귀엽다. 잠깐, 벌써 커플룩 입게 하는 거야? | ③ 강제 |
| Emma | He **doesn't make me wear** them. I want to. | 입으라고 시키는 거 아냐. 내가 입고 싶은 거지. | ③ 강제(부정) |
| Jake | You guys have been dating for like a week. Isn't that a little fast? | 만난 지 일주일밖에 안 됐잖아. 좀 빠르지 않아? | |
| Emma | Depends on the couple, I guess. We like it. He's just really easy to be with, and that alone **makes him a good fit** for me. | 커플마다 다르지 뭐. 우린 좋아. 같이 있기 편하고, 그거 하나만으로도 나한테 딱이야. | ② ~로 만들다 |
| Jake | That's what matters. You know, dating someone kind kind of **makes you a better person**. | 그게 중요하지. 착한 사람 만나면 나도 더 좋은 사람이 되는 거 같아. | ② |
| Emma | Totally. Having him around **has made me** way more patient. | 완전. 걔 덕분에 내가 훨씬 더 참을성 있어졌어. | ② |
| Jake | I get that. Anyway, I'm thinking of getting the new phone, the Pro model. | 이해돼. 아무튼 나 새 폰 살까 해, 프로 모델. | |
| Emma | It's more expensive, though. | 근데 더 비싸잖아. | |
| Jake | True, but the battery life **makes it a better choice** for me. I'm on my phone all day. | 맞아, 근데 배터리 때문에 나한텐 더 나은 선택이야. 하루 종일 폰 하거든. | ② |
| Emma | Fair enough. Oh, before I forget, you promised to **make me** galbi. | 하긴. 아 참, 나 갈비찜 해 주기로 한 거 잊지 마. | ① |
| Jake | I know, I know. Come over Saturday, and I'll **make you** something you won't forget. | 알아 알아. 토요일에 와, 잊지 못할 거 만들어 줄게. | ① |
| Emma | Can't wait. See you then! | 완전 기대돼. 그때 봐! | |

---

### make 세 가지 용법 정리 (이 대화에 다 들어 있음)

- **① make + 사람 + 사물** — "~에게 ~을 만들어/해 주다"  (make you galbi, made me a bracelet)
- **② make + 목적어 + 명사** — "~을 ~로 만들다/되게 하다"  (makes him a good fit, made me more patient)
- **③ make + 사람 + 동사원형** — "(강제로) ~하게 시키다"  (makes me upload, made myself go, make me wear)
